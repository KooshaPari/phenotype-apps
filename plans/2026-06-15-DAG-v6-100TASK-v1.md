## Risks

| Risk | Mitigation |
|---|---|
| Subagents fail to write result files | Track 5 has a 5min polling gate; if files missing, fall back to direct audit |
| `cargo build` errors during migration | Each migration has a verification gate; revert path is git reset |
| Configra is private — can't clone | Skip Configra tasks; document in ADR as "pending owner review" |
| Branch conflicts when pushing | Use `--force-with-lease` only on tracks we own |
| Cross-crate refactor breaks dependents | Run `cargo build --workspace` after each migration step |
