### Track T2 — L52 Encryption-at-Rest Mandate

**Owner:** orchestrator
**Wall-clock:** ~3 days
**LOC:** ~300 (mostly docs + 1 CI check)

Per v18 probe §6 finding 2, **L52 (Encryption-at-Rest)** sits at 1.5 because no fleet-wide policy exists. Substrate crates (pheno-config, pheno-port-adapter) handle secrets in plaintext in memory, no `zeroize` enforcement.

**Deliverables:**

1. `docs/adr/2026-06-21/ADR-078-encryption-at-rest-mandate.md` (200 LOC)
   - Policy: ALL secret material (API keys, tokens, passwords) MUST use `zeroize::Zeroize` on Drop
   - Cargo deny rule: ban `String::from_utf8(secret_bytes)` + similar unsafe patterns
   - Devcontainer + Dockerfile: encrypt swap + tmpfs
   - AWS: enforce EBS encryption + S3 default encryption (terraform module reference)
   - Kubernetes: enforce encryption at rest on PersistentVolumes (NetworkPolicy + StorageClass)

2. `.cargo/audit-rules.toml` (50 LOC) — `cargo-deny` config: deny `unsafe_code` in secret-handling modules (pheno-config, pheno-port-adapter)
3. `pheno-config/src/secrets.rs` — add `#[derive(ZeroizeOnDrop)]` to all 3 secret-holding structs (~50 LOC)

**Score target:** L52 1.5 → 2.0

### Track T5 — L53 Bug Bounty + Pen Test Roadmap

**Owner:** orchestrator
**Wall-clock:** ~2 days
**LOC:** ~280 (docs; ADR + INDEX update)
**Status:** ✅ DONE (2026-06-21, L5-155, ADR-081)

Per v18 probe §6 finding 5, **L53 (Pen Testing)** scored 1/3 (Minimal) because no annual cadence exists and OWASP ASVS 4.0.3 L2 compliance shows 18 of 286 controls failing or untested. This track writes the **roadmap**.

**Deliverables:**

1. `docs/adr/2026-06-21/ADR-081-pentest-roadmap.md` (ADR-081, ~280 LOC)
   - 12-month rolling cadence: 3× internal (quarterly, rotating surfaces) + 1× external (NCC Group annual)
   - Scope: 4 substrate repos (`pheno-mcp-router`, `pheno-port-adapter`, `phenotype-router`, `phenotype-registry`)
   - 4 attack surfaces: A) LLM prompt injection (LlmPort), B) tool-call injection via mcp-router, C) registry poisoning, D) Bifrost plugin deserialization
   - Tooling: semgrep + snyk + nuclei + custom prompt-injection fuzzer + custom Bifrost plugin fuzzer
   - Pass/fail bar: zero Critical, ≤ 2 High
   - Remediation SLA: Critical 24h, High 7d, Medium 30d, Low 90d
   - Budget: $45K/year external (NCC Group) + $20K/year bounty reserve (deferred) = $65K/year
   - Standards: NIST 800-115, OWASP ASVS 4.0.3, OWASP Top 10 for LLM (2025)
   - Vendor: NCC Group as primary external (single-vendor strategy; SOW negotiation target 2026-Q4 for 2027-04-12 kickoff)
   - Bug-bounty launch deferred to Q3 2027 (post-NCC baseline)

2. `docs/adr/2026-06-21/INDEX.md` (updated to add ADR-081)

3. `worklogs/L5-155-v19-T5-pentest-roadmap-2026-06-21.json` (schema v2.1, device=macbook)
