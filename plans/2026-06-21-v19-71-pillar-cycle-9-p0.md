### Track T2 — L52 Encryption-at-Rest Mandate

**Owner:** orchestrator
**Wall-clock:** ~3 days
**LOC:** ~300 (mostly docs + 1 CI check)

Per v18 probe §6 finding 2, **L52 (Encryption-at-Rest)** sits at 1.5 because no fleet-wide policy exists. Substrate crates (pheno-config, pheno-port-adapter) handle secrets in plaintext in memory, no `zeroize` enforcement.

**Deliverables:**

1. `docs/adr/2026-06-21/ADR-078-encryption-at-rest-mandate.md` (200 LOC)
   - Policy: ALL secret material (API keys, tokens, passwords) MUST use `zeroize::Zeroize` on Drop
   - Cargo deny rule: ban `String::from_utf8(secret_bytes)` + similar unsafe patterns
   - Devcontainer + Dockerfile: encrypt swap + tmpfs
   - AWS: enforce EBS encryption + S3 default encryption (terraform module reference)
   - Kubernetes: enforce encryption at rest on PersistentVolumes (NetworkPolicy + StorageClass)

2. `.cargo/audit-rules.toml` (50 LOC) — `cargo-deny` config: deny `unsafe_code` in secret-handling modules (pheno-config, pheno-port-adapter)
3. `pheno-config/src/secrets.rs` — add `#[derive(ZeroizeOnDrop)]` to all 3 secret-holding structs (~50 LOC)

**Score target:** L52 1.5 → 2.0

### Track T3 — L54 OIDC for Service-to-Service

**Owner:** orchestrator + subagent on heavy-runner
**Wall-clock:** ~5 days
**LOC:** ~500 (spec + reference implementation)

Per v18 probe §6 finding 3, **L54 (Federated Identity)** scored 2.0 because ADR-046 exists but no concrete OIDC implementation. This track produces the **reference OIDC client** for federated service-to-service auth.

**Deliverables:**

1. `docs/adr/2026-06-21/ADR-079-oidc-federation-reference.md` (250 LOC)
   - 3 OIDC providers supported: Auth0, Okta, Keycloak
   - JWKS caching + key rotation
   - Token claim → PhenoContext mapping
   - Failure modes (network partition, provider down, expired token)

2. `pheno-context/src/oidc.rs` (200 LOC) — `OidcClient` struct with `verify_id_token()`, `refresh()`, `userinfo()` methods + 3 unit tests
3. `examples/oidc_consumer/` (50 LOC) — example web service that consumes the OIDC bearer token + links to Auth0/Keycloak sandbox

**Score target:** L54 2.0 → 2.5
