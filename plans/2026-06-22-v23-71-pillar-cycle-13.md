## T5 — L42 Macro hygiene (proc-macro span + lint coverage)

### Scope

Reduce unhelpful `compile_error!` sites; add `#[proc_macro_derive]` with proper `Span::call_site()` vs `Span::mixed_site()` hygiene. Each macro crate gets a clippy lint that catches direct `call_site()` use and suggests `mixed_site()` where mixed context is appropriate.

### Files to touch
First identify the 3 proc-macro crates via `rg "proc_macro_derive" --type rust -l`. Likely candidates:
- `pheno-errors-derive/` (if it exists; otherwise pheno-errors)
- `pheno-port-adapter-macros/`
- `pheno-events-macros/`

For each:
- `src/lib.rs` (UPDATE) — audit `Span` usage; replace `call_site()` with `mixed_site()` where mixed context is appropriate
- `src/lib.rs` (UPDATE) — add `#[proc_macro_derive(MyDerive, attributes(my_attr))]` attribute allowlist
- `clippy.toml` (NEW at repo root) — add `forbid-direct-call-site` lint

### Subagent prompt draft
