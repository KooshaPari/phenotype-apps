# v17 — T10 L41 Accessibility (CLI + JSON mode)

**Date:** 2026-06-21
**Cycle:** 7
**Pillar:** L41 (Accessibility)

## Scope

Make 3 CLI surfaces accessible:
1. `pheno-cli-base` — all flags have `--help` descriptions
2. `pheno-flags` — JSON output mode for screen readers
3. `pheno-config` — `--describe-schema` emits machine-readable spec

## Requirements

- Every CLI flag has a `--help` description (min 10 chars)
- `--json` flag emits structured output (not just text)
- ANSI color codes respect `NO_COLOR` env var
- High-contrast mode via `--theme=high-contrast`
- Tabular output has `--no-table` for linear readers

## Implementation pattern

```rust
// cli.rs
#[derive(Args)]
struct CommonOpts {
    /// Output as JSON for machine consumption
    #[arg(long)]
    json: bool,
    /// Disable ANSI colors
    #[arg(long, env = "NO_COLOR")]
    no_color: bool,
    /// Locale for error messages (en, es, ja)
    #[arg(long, default_value = "en")]
    locale: String,
}
```
