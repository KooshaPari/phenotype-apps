## Schedule

All 7 weekly workflows use `cron: '0 16 * * 1'` (Monday 09:00 PDT), matching the existing `chaos.yml` and `lfs-audit-ci.yml` cadence established by ADR-041 (L41 weekly refresh cadence).

## Gate

`evolve-checks.yml` runs on every PR + push to main and to `chore/v3*-**` / `feat/v3*-**` branches. It enforces:
1. All 7 `*-weekly.yml` files exist in `.github/workflows/`
2. Each has a `cron:` schedule line

Failures emit `::error::` annotations on the missing file path.
