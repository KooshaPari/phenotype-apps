# Support

If you need help with Conft, please use one of the channels below.

## Ask a question

- **GitHub Discussions:** https://github.com/KooshaPari/Conft/discussions
- **Bug reports:** https://github.com/KooshaPari/Conft/issues/new?template=bug_report.md
- **Feature requests:** https://github.com/KooshaPari/Conft/issues/new?template=feature_request.md

## Security issues

Please do **not** file public issues for security vulnerabilities.
See [`SECURITY.md`](./SECURITY.md) for the private disclosure policy.

## Contributing

Patches and improvements are welcome — see
[`CONTRIBUTING.md`](./CONTRIBUTING.md) for guidelines.
